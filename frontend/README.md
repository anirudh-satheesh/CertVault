# CertVault Frontend Client

This is the React + Vite frontend client for the CertVault application.

## Quick Start
Before running the frontend, make sure the backend server is running in the background.

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install the necessary node modules:
   ```bash
   npm install
   ```
3. Run the hot-reloading development server:
   ```bash
   npm run dev
   ```

Refer to the main [Workspace README](../README.md) in the project root directory for detailed credentials workspace specifications, prerequisites, and backend server setup.
